## Project Setup

- Please pull the ```master``` branch and run ``` npm install ```.

- Duplicate or Copy ```.env.example``` file and rename it to ```.env```

- Change the ```APP_API_URL``` to backend Laravel project URL

    ```
    default: http://localhost:8000/api
    ```

- Run Project

    ```
     npm run dev 
    ```
